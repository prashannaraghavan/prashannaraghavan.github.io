iConnect

A wearable supported recruiting app

iConnect - Best Wearable Hack - Hack Arizona 2016


Description
An iOS app which streamlines the recruitment process for both Recruiters and Job Seekers. iConnect collects the important keywords from a job description in the form of tags and compare it with the skill set of the candidate and compute a matching percentage. The jobs are suggested to the candidate based on the highest matching percentage. And best candidates are suggested to the employers based on highest matching skill set of the candidate.

Features
User(Job Seeker) of the application would be able to
1. View all the jobs posted in the portal
2. View suggested jobs based on the matching percentage.
3. Save/Apply to the job in one click.
4. Get/View recent job notifications in the Apple Watch and save it for applying in the future.
5. View all the applied jobs.

User(Recruiter) of the application would be able to
1. View all the jobs posted by that company in the portal
2. View all the suggested candidates based on the matching percentage of skills set.
3. View contact details of the best matching candidate.
4. Call/Email candidates in a single click.

Achievement
This app entirely streamlined the manual process of applying to a job and selecting a best candidate for a job by saving significant amount of time for both job seekers and recruiters.
